REQUIREMENTS:
Create a traditional shopping cart where,
 items can be added.
 Items have a quantity, a price each.
Display all items in cart including their descriptions,
quantities and extended price. Plus a grand total.

Entities (Nouns):
    Cart
    Product

Methods (Verbs):
    add_item
    remove_item
    update_quantity
    display_cart

Properties (Adverbs and Adjectives):
    item_price
    item_quantity
    extended_price
    grand_total

Use Case:
    As a user I want to add items to a cart, so I can review them,
    and revise them.
